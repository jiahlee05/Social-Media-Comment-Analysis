## 📊 통계 분석 프롬프트 패턴 (90min)

### 1. t-test (두 그룹 비교)
```
advertising_experiment.csv 데이터로:

Video와 Image 광고 타입의 전환율 차이가
통계적으로 유의미한지 Independent t-test로 확인해줘.

결과에 포함할 것:
- t-statistic, p-value
- 각 그룹의 평균, 표준편차
- 95% 신뢰구간
- 효과 크기 (Cohen's d)
- 결론 (p < 0.05면 유의미)

비전공자도 이해할 수 있게 설명하고,
박스플롯으로 시각화해줘.
```

### 2. ANOVA (여러 그룹 비교)
```
4개 광고 타입 (Image, Video, Carousel, Story)의
전환율을 One-way ANOVA로 비교해줘.

단계:
1. ANOVA 실행 (F-statistic, p-value)
2. 유의미하면 → Tukey HSD 사후 분석
   "어느 그룹과 그룹이 다른가?"
3. 결과를 표로 정리
4. 바이올린 플롯으로 시각화
5. 실무 추천: "어떤 광고 타입에 투자할까?"
```

### 3. 회귀 분석
```
참여도(engagement)가 전환율(conversion_rate)에
미치는 영향을 선형 회귀로 분석해줘.

결과:
- 회귀 방정식 (y = ax + b)
- R² (설명력)
- p-value (유의성)
- 회귀선 포함 산점도
- 해석: "참여도가 1 증가하면 전환율은?"
```

### 4. 다중 회귀
```
여러 변수로 전환율 예측:
독립변수: engagement, cost_per_click, age
종속변수: conversion_rate

각 변수의 회귀 계수와 유의성
모델 R² 및 Adjusted R²
VIF로 다중공선성 확인
가장 영향력 큰 변수는?
```

비전공자도 이해할 수 있게!
