# Session 4: 텍스트 분석 & 워드클라우드

**목표:** 소셜 미디어 댓글을 프롬프트로 분석하고 인사이트 도출

---

## 💬 텍스트 분석 프롬프트 (60min)

### 1. 감성 분석
```
social_media_comments.csv 댓글 데이터를 분석해줘:

1. VADER 또는 다른 감성 분석 도구로
   각 댓글의 감성 점수 계산 (긍정/부정/중립)

2. 전체 감성 분포 (파이 차트)

3. 플랫폼별 감성 비교 (스택 바 차트)

4. 가장 긍정적인 댓글 Top 5
   가장 부정적인 댓글 Top 5

5. 시간에 따른 감성 변화 (선 그래프)

결과를 대시보드 형식으로!
```

### 2. 워드클라우드
```
댓글 텍스트로 워드클라우드 3개 만들어줘:

1. 전체 댓글
   - 크기 1200x600
   - 배경: 흰색
   - 색상: 파란색 계열
   - 불용어 제거 (한국어/영어)

2. 긍정 댓글만
   - 색상: 초록색 계열
   - "만족", "좋아", "최고" 등 강조

3. 부정 댓글만
   - 색상: 빨간색 계열
   - "불만", "나쁨", "최악" 등 강조

각 워드클라우드 아래 Top 10 단어 빈도 표 추가
```

### 3. 모양 커스터마이징
```
브랜드 로고 모양의 워드클라우드를 만들고 싶어.

단계:
1. 간단한 모양 (하트, 별, 원) 마스크 생성
2. 그 모양으로 워드클라우드
3. 윤곽선 강조
4. 고해상도 (300dpi)

또는: "이 이미지 모양으로 만들어줘" + 이미지 첨부
```

---

## 🔍 고급 텍스트 분석 (30min)

### 토픽 모델링
```
댓글 데이터에서 주요 토픽 3-5개를 찾아줘:

방법: LDA 토픽 모델링
각 토픽의:
- 대표 키워드 10개
- 토픽 이름 자동 생성
- 각 토픽이 차지하는 비율
- 시각화 (pyLDAvis 또는 바 차트)

고객들이 주로 어떤 주제에 대해 이야기하는지 요약해줘.
```

---

## 🎯 실습: 고객 리뷰 인사이트 (30min)

**종합 프롬프트:**
```
social_media_comments.csv 분석 리포트:

## 1. 감성 분석
- 전체 긍정/부정/중립 비율
- 플랫폼별 차이
- 통계적 유의성 (카이제곱 검정)

## 2. 핵심 키워드
- 전체 Top 20 워드클라우드
- 긍정/부정 키워드 비교표

## 3. 불만사항 추출
- 부정 댓글에서 자주 언급되는 문제 Top 5
- 각 문제의 빈도
- 대표 댓글 예시

## 4. 개선 제안
- 데이터 기반 액션 아이템 3가지
- 우선순위
- 예상 효과

브랜드 매니저에게 보고할 형식으로!
```

**제출:** 리포트 + 워드클라우드 이미지 + 프롬프트
