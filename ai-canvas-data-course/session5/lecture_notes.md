# Session 5: AI로 대시보드 & 발표자료 생성

**목표:** 분석 결과를 인터랙티브 대시보드와 발표자료로 변환

---

## 📊 대시보드 생성 프롬프트 (60min)

### 1. Plotly Dash 대시보드
```
communication_experiment.csv로 인터랙티브 대시보드를 만들어줘:

구성:
- 상단: 주요 지표 카드 (평균 점수, 참가자 수 등)
- 중간: 조건별 비교 차트 (필터 가능)
- 하단: 상세 데이터 테이블

인터랙티브 기능:
- 드롭다운으로 조건 선택
- 슬라이더로 연령대 필터
- 클릭하면 상세 정보

Plotly Dash 코드 전체 생성해줘.
실행 방법도 알려줘!
```

### 2. Streamlit 앱
```
더 간단한 Streamlit 앱으로 만들어줘:

사이드바:
- 데이터 필터 옵션
- 차트 타입 선택

메인:
- 실시간 업데이트 차트
- 다운로드 버튼 (CSV, PNG)

코드 + 실행 명령어:
"streamlit run app.py"
```

---

## 📽️ 발표자료 생성 (30min)

### PPT 자동 생성 프롬프트
```
분석 결과를 PowerPoint 발표자료로 만들어줘:

슬라이드 구성:
1. 제목 슬라이드
2. Executive Summary (핵심 3가지)
3-5. 주요 분석 결과 (차트 + 해석)
6. 결론 및 추천
7. Q&A

각 슬라이드:
- 차트 이미지
- 간결한 bullet points
- 말할 내용 (발표자 노트)

Python-pptx 코드로 생성해줘.
또는 Markdown으로 내용만 만들어줘.
```

---

## 🎯 최종 프로젝트 (30min)

### 프로젝트 요구사항

**주제 선택:** 제공된 5개 데이터셋 중 1개

**결과물:**
1. **분석 리포트** (Markdown/PDF)
   - Executive Summary
   - 분석 과정
   - 주요 발견
   - 추천사항

2. **대시보드** (Plotly Dash 또는 Streamlit)
   - 인터랙티브 차트
   - 필터 기능
   - 다운로드 옵션

3. **발표자료** (PPT 또는 Markdown)
   - 5-7 슬라이드
   - 시각화 포함
   - 스토리텔링

**종합 프롬프트 예시:**
```
[선택한 데이터셋]으로 완전한 분석 프로젝트:

## 1. 데이터 분석
- EDA
- 통계 검정
- 시각화 (5개 이상)

## 2. Streamlit 대시보드
- 필터 기능
- 인터랙티브 차트
- 다운로드 버튼

## 3. 발표자료 (Markdown)
- 슬라이드 7장
- 스토리텔링 구조
- 차트 포함

## 4. 프롬프트 문서화
- 사용한 모든 프롬프트 기록
- 개선 과정
- 배운 점

모든 것을 한 번에 생성해줘!
```
