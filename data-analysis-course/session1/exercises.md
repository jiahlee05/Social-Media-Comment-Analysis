## 💡 학습 팁

### AI 활용 전략
1. **막히면 즉시 질문**: "이 오류가 왜 나는지 설명해줘"
2. **코드 리뷰 요청**: "이 코드를 더 효율적으로 만들 수 있을까?"
3. **개념 이해**: "GroupBy가 정확히 무엇을 하는 건지 예제와 함께 설명해줘"
4. **디버깅**: 오류 메시지를 그대로 AI에게 보여주기

### 자주 하는 실수
❌ `platfrom` (오타) → ✅ `platform`
❌ `df.groupby('platform').mean()` (모든 컬럼 평균) → ✅ `df.groupby('platform')['likes'].mean()`
❌ 0으로 나누기 → ✅ 분모에 +1 또는 조건 체크

### 도움이 되는 Pandas 치트시트
```python
# 자주 쓰는 명령어
df.head(n)              # 처음 n행
df.tail(n)              # 마지막 n행
df.shape                # (행, 열) 개수
df.columns              # 컬럼 이름
df.info()               # 데이터 정보
df.describe()           # 기술 통계

df['col'].value_counts()# 빈도수
```
