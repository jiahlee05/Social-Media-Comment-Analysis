## 🎯 데이터 분석 마스터 프롬프트

### 완전한 EDA (탐색적 데이터 분석)
```
[파일명].csv 파일을 첨부했어.

완전한 탐색적 데이터 분석(EDA)을 수행해줘:

## 1. 데이터 개요
- 행/열 개수
- 컬럼별 데이터 타입
- 결측값 현황 (시각화 포함)
- 처음 10개 행

## 2. 기술 통계
- 수치형: 평균, 중앙값, 표준편차, 사분위수
- 범주형: 고유값, 빈도
- 표로 정리

## 3. 분포 분석
- 각 수치형 변수의 히스토그램
- 박스플롯으로 이상치 확인

## 4. 관계 분석
- 변수 간 상관관계 히트맵
- 주요 관계 산점도

## 5. 범주형 분석
- 카테고리별 분포
- 교차 분석

## 6. 인사이트
- 주요 발견 5가지
- 이상한 점이나 흥미로운 패턴
- 추가 분석 제안

전문적인 스타일로, 비전공자도 이해할 수 있게!
```

## 📊 시각화 마스터 프롬프트

### ggplot 수준 대시보드
```
[데이터명]으로 전문가급 분석 대시보드를 만들어줘:

레이아웃: 2x3 subplot (총 6개 차트)

차트 1 (상단 왼쪽): [변수] 분포
- 히스토그램 + KDE
- 평균/중앙값선 표시

차트 2 (상단 중간): 카테고리 비교
- 그룹 막대그래프
- 오차 막대 포함

차트 3 (상단 오른쪽): 시간 트렌드
- 선 그래프
- 마커 + 신뢰구간

차트 4 (하단 왼쪽): 관계 분석
- 산점도 + 회귀선
- 상관계수 표시

차트 5 (하단 중간): 분포 비교
- 바이올린 또는 박스플롯
- 그룹별 색상 구분

차트 6 (하단 오른쪽): 상관관계
- 히트맵
- 값 표시

전체 스타일:
- 색상 테마: [파란색/초록색/회색 등]
- 배경: 흰색
- 그리드: 연한 회색
- 폰트: 깔끔한 sans-serif
- ggplot2 스타일

전체 제목: "[분석 주제] 종합 대시보드"
각 차트에 간단한 해석 추가
```

## 📈 통계 분석 마스터 프롬프트

### 완전한 통계 리포트
```
[데이터셋]으로 통계 분석 리포트를 작성해줘:

## 1. 기술 통계
- 전체 요약
- 그룹별 통계 (평균, 표준편차, 최소, 최대)

## 2. 정규성 검정
- Shapiro-Wilk test
- Q-Q plot
- 결과 해석

## 3. 그룹 비교
[2개 그룹인 경우]
- Independent t-test
- Mann-Whitney U (비모수)
- Cohen's d (효과 크기)

[3개 이상 그룹인 경우]
- One-way ANOVA
- Kruskal-Wallis (비모수)
- Tukey HSD 사후 분석
- 효과 크기 (eta-squared)

## 4. 상관/회귀 분석
- Pearson/Spearman 상관계수
- 선형 회귀 (R², 회귀식)
- 잔차 분석

## 5. 시각화
- 그룹별 박스플롯
- 회귀선 포함 산점도
- 평균 비교 바 차트 (오차 막대)

## 6. 결론
- 통계적 유의성 요약
- 실무적 의미
- 한계점

비전공자도 이해할 수 있게, 표와 차트 적극 활용!
```

## 💬 텍스트 분석 마스터 프롬프트

### 종합 텍스트 분석
```
[텍스트 데이터셋]을 완전히 분석해줘:

## 1. 전처리
- 소문자 변환
- 특수문자/이모지 제거
- 토큰화
- 불용어 제거 ([한국어/영어])
- 전처리 전후 비교

## 2. 빈도 분석
- Top 30 단어 빈도
- 막대그래프
- Bigram (2단어 조합) 분석

## 3. 워드클라우드
세 가지 버전:
a) 전체 텍스트
b) 긍정 텍스트만
c) 부정 텍스트만

크기: 1200x600
색상: 테마에 맞게
고해상도 (300dpi)

## 4. 감성 분석
- VADER 또는 유사 도구
- 긍정/부정/중립 분류
- 플랫폼별/시간별 감성 비교
- 감성 점수 분포 히스토그램
- 통계 검정 (그룹 간 감성 차이)

## 5. 토픽 모델링
- LDA로 주요 토픽 3-5개
- 각 토픽의 대표 키워드
- 토픽 분포 시각화

## 6. 인사이트
- 주요 발견 5가지
- 긍정 의견 요약
- 부정 의견 요약
- 개선 제안 3가지

대시보드 형식으로 정리!
```

## 🎨 Plotly 인터랙티브 차트

### 완벽한 인터랙티브 대시보드
```
Plotly를 사용해서 완전한 인터랙티브 대시보드를 만들어줘:

데이터: [파일명]

구성:
1. 드롭다운 메뉴: 카테고리 선택
2. 슬라이더: 시간 범위 선택
3. 체크박스: 데이터 시리즈 on/off

차트 3개:
- 시계열 선 그래프 (줌/팬 가능)
- 카테고리 막대그래프 (클릭으로 드릴다운)
- 산점도 (hover로 상세 정보)

모든 차트 연동:
- 한 차트에서 선택하면 다른 차트도 업데이트
- 필터 적용시 전체 반영

추가 기능:
- 다운로드 버튼 (PNG, SVG)
- 데이터 테이블 (정렬/필터 가능)
- 요약 통계 카드

실행 가능한 HTML 파일로 저장
```

## 📊 대시보드 생성 마스터 프롬프트

### Streamlit 완전한 앱
```
Streamlit으로 데이터 분석 웹앱을 만들어줘:

## 사이드바
- 파일 업로드 (CSV)
- 필터 옵션:
  * 카테고리 선택 (멀티셀렉트)
  * 날짜 범위 (슬라이더)
  * 수치 범위 (슬라이더)

## 메인 페이지

### Tab 1: 데이터 개요
- 전체 통계 요약
- 샘플 데이터 미리보기
- 결측값 현황

### Tab 2: 시각화
- 차트 타입 선택 (radio)
- 동적 업데이트 차트
- 다운로드 버튼

### Tab 3: 통계 분석
- 검정 방법 선택
- 결과 표시
- 해석 텍스트

### Tab 4: 리포트 생성
- 분석 요약
- PDF 다운로드
- 이메일 전송 (선택)

전체 코드 + 실행 방법
requirements.txt도 포함
```

```

## Part 1: 분석 리포트 (Markdown)

### Executive Summary
- 핵심 발견 3가지
- 주요 추천사항

### 1. 서론
- 배경 및 목표
- 연구 질문 3개

### 2. 데이터
- 데이터 설명
- 전처리 과정
- 데이터 품질

### 3. 분석 방법
- 사용한 기법
- 왜 선택했는지

### 4. 결과
- 기술 통계
- 통계 검정
- 시각화 (최소 5개)
- 각 결과의 해석

### 5. 논의
- 주요 발견
- 실무적 의미
- 한계점

### 6. 결론
- 요약
- 추천사항
- 향후 연구

## Part 2: 인터랙티브 대시보드
- Plotly Dash 또는 Streamlit
- 필터 기능
- 다운로드 옵션
- 실행 가능한 코드

## Part 3: 발표자료 (Markdown 슬라이드)
- 7-10 슬라이드
- 차트 포함
- 발표자 노트

## Part 4: 코드 패키지
- 재사용 가능한 함수들
- 주석 완벽
- README 포함

모든 것을 한 번에 생성하고
각 파일을 구분해서 제공해줘!
```
