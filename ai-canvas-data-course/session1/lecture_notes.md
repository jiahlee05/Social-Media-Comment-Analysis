## 1. AI 캔버스 3종 비교 (30분)

### 왜 AI 캔버스인가?

**기존 방식의 문제점:**
```python
# ❌ 이런 코드를 외우고... 타이핑하고... 디버깅하고...
import pandas as pd
df = pd.read_csv('data.csv')
result = df.groupby('category')['value'].mean()
# 에러 발생... 구글링... 스택오버플로우...
```

**AI 캔버스 방식:**
```
✅ 프롬프트 입력:
"data.csv를 읽어서 카테고리별 평균값을 계산하고
막대그래프로 시각화해줘"

→ AI가 코드 생성 + 실행 + 차트 표시
→ 수정 필요하면 "색상 변경", "제목 추가" 등 자연어로!
```

## 2. Claude Artifacts로 첫 분석 (40분)

### 단계 1: Claude Artifacts 시작하기

**접속:** https://claude.ai

**Artifacts란?**
- Claude와 대화하면서 생성된 코드/차트/문서가 **별도 창**에 표시
- 실시간으로 수정 가능
- HTML/React/SVG로 인터랙티브 시각화

### 단계 2: 데이터 업로드 방법

#### 방법 A: 파일 업로드 (추천)
```
프롬프트:
"📎 (파일 첨부 아이콘 클릭) social_media_engagement.csv

이 데이터를 분석해줘:
1. 어떤 컬럼들이 있는지 확인
2. 기본 통계 요약
3. 처음 10개 행 보여줘"
```

#### 방법 B: 데이터 붙여넣기
```
프롬프트:
"다음은 소셜 미디어 데이터야:

post_id,platform,likes,shares
1,Instagram,850,120
2,Twitter,620,90
...

이 데이터를 분석해줘"
```

### 단계 3: 탐색적 데이터 분석 (EDA) 프롬프트

**🎯 목표: 데이터 전체 파악하기**

```
프롬프트 (복사해서 사용):

이 소셜 미디어 데이터를 탐색적으로 분석해줘:

1. 데이터 개요:
   - 총 행 수와 열 수
   - 각 컬럼의 데이터 타입
   - 결측값이 있는지 확인

2. 기술 통계:
   - 수치형 변수의 평균, 중앙값, 표준편차
   - 범주형 변수의 고유값과 빈도

3. 주요 인사이트:
   - 어떤 플랫폼이 가장 많은 포스트를 가지는지
   - 좋아요와 공유 수의 관계
   - 이상치가 있는지

4. 시각화:
   - 플랫폼별 평균 좋아요 수 막대그래프
   - 좋아요 vs 공유 산점도

결과를 보기 쉽게 정리해서 보여줘!
```

**Claude의 응답 예시:**
- Artifacts 창에 인터랙티브 차트 표시
- 채팅창에 분석 요약 및 해석
- 필요시 코드도 함께 표시

### 단계 4: 시각화 개선하기

**기본 차트가 생성되면, 자연어로 개선:**

```
개선 프롬프트 예시:

"차트를 개선해줘:
- 막대 색상을 파란색 계열로
- 제목을 '플랫폼별 평균 참여도 비교'로
- 각 막대 위에 정확한 수치 표시
- 배경을 흰색으로
- 더 전문적으로 보이게"
```

**또는:**
```
"이 차트를 Plotly로 인터랙티브하게 만들어줘.
마우스 오버하면 상세 정보가 뜨도록"
```

### 단계 5: 인사이트 요청하기

```
인사이트 프롬프트:

"이 분석 결과에서:
1. 가장 중요한 발견 3가지
2. 소셜 미디어 전략 추천 3가지
3. 추가로 분석하면 좋을 것들

커뮤니케이션 전공 학생이 이해할 수 있게 설명해줘"
```

## 3. ChatGPT Canvas로 개선하기 (30분)

### Canvas 활성화 방법

**접속:** https://chat.openai.com

1. 새 채팅 시작
2. 프롬프트 입력란 옆의 **📄 Canvas 아이콘** 클릭
3. 또는 프롬프트에 "Canvas를 사용해서..."라고 명시

### Canvas의 강점: 코드 편집

**프롬프트:**
```
Canvas를 사용해서 social_media_engagement.csv 데이터를
분석하는 Python 코드를 작성해줘.

포함사항:
1. 데이터 로드
2. 기술 통계 출력
3. 플랫폼별 평균 좋아요 시각화
4. 주석을 자세히 달아줘
```

**Canvas 화면 구성:**
```
┌─────────────┬──────────────────┐
│             │  Canvas (코드)    │
│   대화창    │  ✏️ 직접 편집 가능 │
│             │  💾 저장           │
│             │  📋 복사           │
└─────────────┴──────────────────┘
```

### Canvas 활용 팁

1. **코드 설명 요청:**
   - Canvas 코드 선택 → "이 부분 설명해줘"

2. **부분 수정:**
   - "10번째 줄만 수정해줘: 색상을 빨간색으로"

3. **버전 비교:**
   - Canvas 하단의 버전 히스토리로 이전 버전 비교

4. **다른 언어로 변환:**
   - "이 Python 코드를 R로 변환해줘"

## 4. 실전: 소셜 미디어 데이터 분석 (20분)

### 실습 과제: 30분 챌린지

**미션:** Claude Artifacts만 사용해서 완전한 분석 리포트 만들기

**데이터:** `social_media_engagement.csv`

**요구사항:**

1. **데이터 탐색**
   - 데이터 구조 파악
   - 기술 통계

2. **핵심 질문 답하기**
   - 어떤 플랫폼이 가장 높은 참여도?
   - 연령대별 선호 플랫폼은?
   - 포스트 시간대와 참여도의 관계는?

3. **시각화 (최소 3개)**
   - 플랫폼별 비교 차트
   - 시간대별 트렌드
   - 자유 선택 1개

4. **인사이트 및 추천**
   - 주요 발견사항 3가지
   - 전략 제안 3가지

### 마스터 프롬프트 (한 번에!)

```
social_media_engagement.csv 파일을 첨부했어. (📎 파일 첨부)

커뮤니케이션 학과 학생을 위한 완전한 분석 리포트를 만들어줘:

## 1. 데이터 개요
- 구조 설명
- 기술 통계
- 데이터 품질 (결측값, 이상치)

## 2. 핵심 분석
### 질문 1: 플랫폼별 성과
- 어떤 플랫폼이 가장 높은 좋아요/공유/댓글을 받나?
- 통계적으로 유의미한 차이가 있나?
- 막대그래프로 시각화

### 질문 2: 연령대 패턴
- 각 연령대가 선호하는 플랫폼은?
- 참여도 차이는?
- 히트맵 또는 그룹 막대그래프로

### 질문 3: 시간 전략
- 언제 포스트하면 참여도가 높은가?
- 시간대별 트렌드
- 선 그래프로

## 3. 추가 인사이트
- 상관관계 분석 (좋아요-공유-댓글)
- 히트맵 시각화

## 4. 결론 및 추천
- 핵심 발견 3가지
- 소셜 미디어 전략 3가지
- 실행 가능한 액션 아이템

모든 차트는:
- 전문적인 디자인
- 명확한 제목과 레이블
- 색상 일관성 (파란색 계열)
- 해석 포함

비전공자도 이해할 수 있게 작성해줘!
```

**예상 소요 시간:** 5-10분 (AI가 생성)
**학생 작업:** 결과 검토, 필요시 개선 요청
