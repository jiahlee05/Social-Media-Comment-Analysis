# AI 캔버스로 배우는 데이터 분석
## 프롬프트만으로 완성하는 전문가급 분석 & 시각화

**대상:** 겐트 글로벌 캠퍼스 / 유타대 커뮤니케이션 학과 신입생
**수업 시간:** 주 1회, 2시간 (총 5회)
**핵심 철학:** "코딩은 참고만, 실제는 AI 캔버스로!"

---

## 🎯 이 과정이 특별한 이유

### 기존 방식 ❌
```python
# 복잡한 코드를 외우고 디버깅...
import pandas as pd
import matplotlib.pyplot as plt
df = pd.read_csv('data.csv')
df.groupby('category')['value'].mean().plot(kind='bar')
plt.title('Average by Category')
plt.show()
```

### 새로운 방식 ✅
```
프롬프트 입력:
"data.csv에서 카테고리별 평균값을 계산하고,
전문적인 막대그래프로 시각화해줘.
색상은 브랜드 컬러(파란색 계열)로,
제목과 레이블도 명확하게."

→ AI가 즉시 완벽한 차트 생성!
→ 수정도 자연어로!
```

## 💡 핵심 컨셉

### 1. **노코드/로우코드 분석**
- Python/R 문법은 "이런게 있구나" 수준만 이해
- 실제 작업은 AI 캔버스로 프롬프트만 입력
- ggplot/plotly 갤러리 수준의 결과물을 즉시 생성

### 2. **AI 캔버스 3종 마스터**

- ggplot2/Seaborn 갤러리 둘러보기

### **Session 3: 통계 분석 프롬프트 마스터** (2시간)
**목표:** 통계 검정을 프롬프트로 실행하고 결과 해석하기

**내용:**
- "통계적으로 유의미한가?" 질문하기
- t-test, ANOVA 프롬프트 템플릿
- 회귀 분석 & 상관관계
- 결과 해석을 자연어로 요청

**프롬프트 예시:**
```
"이 광고 데이터에서 Video와 Image 타입의
전환율 차이가 통계적으로 유의미한지 t-test로 확인해줘.
결과를 비전공자도 이해할 수 있게 설명해줘."
```

**실습:** A/B 테스트 분석
- 실험군-대조군 비교
- 효과 크기 계산
- 의사결정 리포트 생성

## 💡 학습 팁

### AI 캔버스 활용 전략
1. **구체적으로 요청:** "차트 만들어줘" ❌ → "파란색 막대그래프, 제목은 '플랫폼별 참여도', 값 레이블 포함" ✅
2. **반복 개선:** 한 번에 완벽하지 않아도 OK → "색상 더 밝게", "범례 위치 변경" 등으로 다듬기
3. **예시 제공:** 원하는 스타일의 이미지나 링크 공유
4. **오류 시:** 에러 메시지를 그대로 AI에게 보여주기

### 프롬프트 작성 팁
```
좋은 프롬프트 구조:

1. 맥락 제공: "나는 커뮤니케이션 전공이고..."
2. 데이터 설명: "이 CSV는 소셜 미디어 참여도 데이터야"
3. 원하는 결과: "플랫폼별 평균 좋아요 수를 비교하고 싶어"
4. 스타일 요구: "전문적인 발표자료에 쓸 거야"
5. 추가 요청: "해석도 간단히 덧붙여줘"
```
