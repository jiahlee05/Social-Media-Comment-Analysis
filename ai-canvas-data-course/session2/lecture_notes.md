## 📊 고급 시각화 프롬프트 (50min)

### 1. 복합 대시보드
```
2x2 subplot으로 한 화면에 4개 차트:
1. 플랫폼별 평균 (막대)
2. 시간 트렌드 (선)
3. 상관관계 (히트맵)
4. 분포 (바이올린)

전체 제목: "종합 분석 대시보드"
통일된 색상 테마: 파란색 계열
```

### 2. Plotly 인터랙티브
```
Plotly로 인터랙티브 산점도:
- 마우스 오버시 모든 정보 표시
- 플랫폼별 색상 구분
- 줌/팬 가능
- 범례 클릭으로 필터링
- 다운로드 버튼
```

### 3. 애니메이션
```
시간에 따른 변화를 보여주는 애니메이션 차트:
- X축: 시간
- Y축: 참여도
- 플랫폼별로 다른 색상
- 자동 재생, 일시정지 버튼
```

## 🎯 실습: 뉴스 소비 패턴 시각화 (40min)

**데이터:** `news_consumption.csv`

**미션:** ggplot 갤러리 수준의 차트 3개 생성

**마스터 프롬프트:**
```
news_consumption.csv로 전문가급 시각화 3개 만들어줘:

1. 시간대별 뉴스 소비 트렌드
   - Plotly 인터랙티브 선 그래프
   - 디바이스별로 다른 선
   - 마우스 오버시 정확한 수치
   - ggplot 스타일

2. 연령대-디바이스 히트맵
   - Seaborn 스타일
   - 각 셀에 평균 독서 시간
   - 색상: Blues 팔레트
   - 주석 포함

3. 3D 산점도 (선택사항)
   - X: 나이
   - Y: 독서 시간
   - Z: 참여도 점수
   - 색상: 디바이스
   - Plotly로 회전 가능

모든 차트: 발표 품질, 고해상도, 일관된 테마
```

**과제:**  프롬프트만으로 30분 안에 완성! 📊
