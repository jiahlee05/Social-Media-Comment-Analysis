## 💡 통계 검정 선택 가이드

| 상황 | 사용할 검정 |
|------|-------------|
| 두 집단 비교 (독립) | Independent t-test |
| 두 집단 비교 (대응) | Paired t-test |
| 3개 이상 집단 비교 | ANOVA → Tukey HSD |
| 두 변수 관계 | Correlation, Regression |
| 비정규 데이터 (2집단) | Mann-Whitney U |
| 비정규 데이터 (3+집단) | Kruskal-Wallis |
